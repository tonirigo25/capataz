# Perfiles sectoriales

Orqena incluye: servicios generales, construcción, instalaciones, servicios profesionales, consultoría, agencia, taller de reparación, servicios de salud, educación, comercio, hostelería, inmobiliario y otro.

Cada perfil define nombre, descripción, términos de trabajo, responsable y progreso, ejemplos, icono, acciones rápidas, tipos sugeridos, campos, KPIs, documentos, agenda y sugerencias. `terminologyOverrides` permite ajustes por empresa.

El perfil sanitario solo adapta lenguaje administrativo: no incluye historia clínica, datos médicos ni afirmaciones de cumplimiento sanitario. No se renombran modelos históricos como `Work` ni se actualizan tenants mediante SQL.
