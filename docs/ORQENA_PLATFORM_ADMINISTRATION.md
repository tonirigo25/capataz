# Administración de plataforma

`/plataforma` está reservada a `PLATFORM_OWNER`, `PLATFORM_ADMIN`, `PLATFORM_SUPPORT` y `PLATFORM_ANALYST`. Estos roles no se reutilizan dentro de empresas. La vista ofrece salud comercial y operaciones administrativas sin exponer facturas, documentos, clientes, conversaciones, memoria ni datos bancarios.

El primer PLATFORM_OWNER solo se configura localmente mediante `npm run commercial:setup-local -- --platform-owner-email <email>` con confirmación explícita de base local.
