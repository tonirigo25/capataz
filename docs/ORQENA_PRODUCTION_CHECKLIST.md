# Checklist de producción

- [ ] Autorización expresa de promoción y SHA.
- [ ] Proyecto, servicio, entorno, dominio y PostgreSQL identificados.
- [ ] Snapshot/PITR probado.
- [ ] Migraciones revisadas y aditivas.
- [ ] Secretos independientes y proveedores aprobados.
- [ ] Correo, billing, webhooks y OpenAI configurados o degradados explícitamente.
- [ ] DNS, canonical, robots y sitemap con política de producción.
- [ ] E2E, visual, accesibilidad, suites, typecheck, build y runner verdes.
- [ ] Observabilidad, alertas y rollback asignados.
- [ ] Staging no comparte proyecto ni servicio lógico con production.
