# Guía de contenido de Orqena

## Tono

Directo, sereno, profesional y orientado a la siguiente tarea. Frases cortas, verbos concretos y vocabulario del negocio. Explicar consecuencias antes de pedir confirmación.

## Mensajes

- Error: qué ocurrió, qué se conservó y cómo continuar.
- Vacío: qué falta y una acción útil.
- Confirmación: empresa, acción, registros, importes, impuestos, fechas y consecuencias.
- Transparencia: indicar fuente o incertidumbre sin mostrar prompts, modelos, JSON ni razonamiento interno.

## Palabras prohibidas en interfaz

No mostrar nombres de fases, Prisma, OpenAI, algoritmos, “señales deterministas”, “flujo seguro”, “repositorio documental”, “placeholders soportados” ni notas de ingeniería.

Incorrecto: “Señal determinista detectada.” Correcto: “Requiere atención.”

Incorrecto: “Faltan datos.” Correcto: “Faltan cuatro datos para emitir documentos completos.”
